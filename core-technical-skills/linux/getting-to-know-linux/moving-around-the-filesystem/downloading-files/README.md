# Downloading Files

Downloading files is a essential requirement, given the online nature of the world today. Therefore, having the ability to perform this action from within a shell environment is vital. We will look at three main tools to achieve this:

* [axel](https://github.com/axel-download-accelerator/axel)
* [wget](https://www.gnu.org/software/wget/)
* [curl](https://curl.se/)
