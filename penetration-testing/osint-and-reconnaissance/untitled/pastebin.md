# Pastebin

[Pastebin](https://pastebin.com/) is a website for storing and sharing text. Note that the site **doesn’t require an account for basic usage**. Many people \(and administrators...\) use Pastebin because it is ubiquitous and simple to use. But since Pastebin is a public service, we can use it to search for sensitive information:

We can use the website for basic searches, or use the API for more advanced uses. However, a quick an effective way to search Pastebin is via Google Dorks:

{% hint style="info" %}
Note that a lot of the results that come back from Pastebin are full of random characters. This is often because the link that Pastebin generates is made of random characters, but often its because something "dodgy" has been posted there, and the originator is trying \(and failing\) to keep their paste discreet
{% endhint %}
